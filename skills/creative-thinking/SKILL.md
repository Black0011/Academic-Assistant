# Creative Thinking for Research — 创意思维框架

> 适配自 orchestra-research/creative-thinking-for-research。

与 brainstorming Skill 的关系：brainstorming 提供操作性工作流（发散→收敛→精炼）；本 Skill 提供更深层的认知引擎来驱动创造性跳跃。两者组合使用效果最佳。

## 8 个认知科学框架

### 1. 组合创造（Bisociation）

将两个无关领域的概念交叉组合。不是表面比喻（"网络像大脑"），而是结构映射（"注意力机制实现了认知科学中选择性注意的门控机制"）。

**工作流**：选两个领域 → 列出各自的核心原语 → 创建交叉矩阵 → 找有意义的交叉点。

### 2. 问题重构

从"如何解决这个问题"切换到"我是否在正确地思考这个问题"。

| 重构策略 | 示例 |
|---------|------|
| 改变目标 | "如何加速算法" → "如何消除对这个计算的需要" |
| 改变形式化 | 图问题 → 线性代数问题（谱方法） |
| 改变粒度 | 逐 token 预测 → 逐 span 预测 |
| 改变视角 | "模型如何学习" → "数据如何教学"（课程学习） |

### 3. 类比推理（Structure-Mapping）

Gentner 的结构映射理论：远距离类比比近距离类比更有突破性，但需要更严格的结构验证。

**关键**：用无术语的关系语言描述问题，然后搜索其他领域的结构匹配。

### 4. 约束操纵（Boden 框架）

| 创造力类型 | 操作 | 示例 |
|-----------|------|------|
| 探索性 | 在现有范式内搜索 | 超参数优化 |
| 组合性 | 组合不同空间的元素 | 多任务学习 + 神经符号 |
| 变革性 | 改变空间本身的规则 | 放弃"训练需要标签"（自监督学习） |

列出当前方法的所有约束，分为硬（物理/逻辑必须）、软（惯例）、隐（未明说的假设），然后专攻软/隐约束。

### 5. 否定与反转

取领域内的核心假设，逆转它。

| 假设 | 否定 | 产物 |
|------|------|------|
| "训练和推理是分离的" | 如果不分呢？ | 在线学习、测试时训练 |
| "需要精确答案" | 如果近似就够？ | 近似算法、草图数据结构 |
| "更多参数=更多计算" | 如果不激活所有参数？ | 混合专家模型 |

### 6. 抽象与泛化阶梯

- **泛化**：当你有结果但没有解释时
- **特化**：当你有理论但没有落地时
- **类比**：当你在两个方向都卡住时

### 7. 邻近可能（Adjacent Possible）

创新发生在当前可达的边界——最近 1-2 年什么新条件出现了？这些新条件让什么之前不可行的方向变得可行了？

### 8. 雅努斯式思维

持有两个看似矛盾的观点，不急于选边，而是寻求超越对立的综合框架。

## 快速使用协议

**被卡住时**（15 分钟）：
1. 约束操纵（F4）：列出所有隐性约束
2. 否定（F5）：反转 3 个软约束
3. 邻近可能（F7）：最近什么变了？

**需要全新视角时**（30 分钟）：
1. 组合创造（F1）：交叉矩阵
2. 问题重构（F2）：换 3 种表述方式
3. 类比推理（F3）：找远距离结构匹配
